# Python Visualizations in Power BI

## Overview
This project integrates Python scripts within Power BI to perform advanced analysis and create custom visualizations using the `bike_sales_data` dataset.

## Features
- Python scripting enabled in Power BI.
- Visualizations using Matplotlib and Seaborn.
- Trend analysis (Registration Year vs. Resale Price).
- KPI-style comparisons (Average vs. Current Mileage).
- Category comparisons (Revenue by Product Brand).
